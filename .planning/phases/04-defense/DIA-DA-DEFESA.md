## Logística (dia da defesa)

- [ ] Chegar à **sala** pelo menos **15 min antes**; ao campus, 45 min antes.
- [ ] Testar **projetor** com o notebook principal (cabo/adaptador HDMI — levar o adaptador!).
- [ ] Testar **áudio** se houver microfone; falar uma frase de teste do fundo da sala.
- [ ] **Slides já abertos** em tela cheia antes de começar (PDF, não editor).
- [ ] Terminal com o clone local preparado + browser com o repo GitHub em aba (ver `DEMO-CHECKLIST.md`).
- [ ] Notebook: carregador conectado, notificações desativadas (modo não perturbe), brilho no máximo.
- [ ] Relógio/celular do cronometrista visível (checkpoints 5/10/13 min do `ROTEIRO.md`).

## Postura (VAL-04 / VAL-05)

- Falar **para a banca, nunca para o slide**. O slide fica atrás; se precisar apontar, apontar e voltar o olhar.
- Frases curtas. Terminou a ideia? Pare. Não preencher silêncio com "né", "então", "tipo".
- Um integrante fala por vez; os demais olham para quem fala (não para o celular/chão).
- Volume: quem está no fundo da sala precisa ouvir sem esforço.
- Transições combinadas: quem termina anuncia o próximo ("o [Nome] apresenta agora os resultados").
